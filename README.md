## About

A reusable chart component library for wireframing dashboards in Excalidraw.

Use it to quickly sketch dashboard ideas before moving into a full build. Once imported, the components will appear in your Excalidraw library panel.

<br>

## Quick start

1. Get the library file: [`datavizfairy-chart-library.excalidrawlib`](./datavizfairy-chart-library.excalidrawlib)
2. Open [Excalidraw](https://excalidraw.com/)
3. Open the Library panel
4. Click the three dots menu
5. Select **Open**
6. Choose the `.excalidrawlib` file

The chart library should then appear in your Excalidraw library.

<br>

## What’s included

The library includes reusable components for:

* Bar charts
* Line charts
* Donut charts
* Scatterplots
* Stacked bars
* Box plots
* Sankey charts
* Maps
* Additional dashboard wireframing elements
